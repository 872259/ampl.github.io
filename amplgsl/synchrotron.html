Synchrotron Functions¶

gsl_sf_synchrotron_1(x)¶

This routine computes the first synchrotron function \(x \int_x^\infty K_{5/3}(t) dt\) for \(x \geq 0\).

gsl_sf_synchrotron_2(x)¶

This routine computes the second synchrotron function \(x K_{2/3}(x)\) for \(x \geq 0\).
